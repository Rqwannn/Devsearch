# DevSearch

Aplikasi Python Pertama

# Installation

- 1 - clone repositori
- 2 - cd ke project "cd devsearch"
- 3 - buat virtual environment dan activate
- - pip install virtualenv
- - virtualenv envname
- - envname\scripts\activate
- 4 - pip install -r requirements.txt
- 5 - python manage.py runserver
